# 商赛 PPT 风格 Skill

用于制作中国商业比赛、行业分析、商业提案与答辩场景的 PPTX 或 HTML。

## 调用

```text
用商赛 PPT 风格制作：[主题]
输出格式：[PPTX / HTML]
页数：[页数]
受众：[评委 / 客户 / 管理层 / 课堂]
目标：[分析 / 提案 / 路演 / 答辩]
```

以下表达会直接触发本 Skill：

- 商赛 PPT 风格
- 用商赛 PPT 风格
- 按照商赛 PPT 风格做
- 商赛风格 PPT

## 安装

```bash
python3 "$HOME/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py" \
  --repo victorpaulee/Playground \
  --path skills/chinese-business-slides
```

安装后重启 Codex。
